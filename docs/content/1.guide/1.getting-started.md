# Getting Started

Get up and running with v-maplibre in your Vue 3 or Nuxt application.

## Installation

Install the package and its peer dependency:

::code-group

```bash [pnpm]
pnpm add @geoql/v-maplibre maplibre-gl
```

```bash [npm]
npm install @geoql/v-maplibre maplibre-gl
```

```bash [yarn]
yarn add @geoql/v-maplibre maplibre-gl
```

::

## Import Styles

Import the MapLibre GL CSS in your main application file:

```typescript
// main.ts or app.vue
import 'maplibre-gl/dist/maplibre-gl.css';
```

## Basic Usage

Here's a simple example to render a map:

```vue
<script setup lang="ts">
  import { VMap } from '@geoql/v-maplibre';
  import 'maplibre-gl/dist/maplibre-gl.css';

  const mapOptions = {
    style: 'https://demotiles.maplibre.org/style.json',
    center: [-74.5, 40],
    zoom: 9,
  };
</script>

<template>
  <VMap :options="mapOptions" style="height: 500px"></VMap>
</template>
```

## Nuxt Usage

For Nuxt applications, you need to configure the component to only render on the client side:

```vue
<script setup lang="ts">
  import { VMap } from '@geoql/v-maplibre';

  const mapOptions = {
    style: 'https://demotiles.maplibre.org/style.json',
    center: [-74.5, 40],
    zoom: 9,
  };
</script>

<template>
  <ClientOnly>
    <VMap :options="mapOptions" style="height: 500px"></VMap>
  </ClientOnly>
</template>
```

### Nuxt Plugin (Optional)

You can also create a Nuxt plugin to globally import the styles:

```typescript
// plugins/maplibre.client.ts
import 'maplibre-gl/dist/maplibre-gl.css';

export default defineNuxtPlugin(() => {
  // Plugin loaded
});
```
